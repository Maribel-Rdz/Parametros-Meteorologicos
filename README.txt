Medición de parámetros meteorológicos
------------------------------------------
Este trabajo consiste en monitorear dichos parámetros en tiempo real a través de una
página web, con el fin de indicarnos si la planta se encuentra estable.

Parámetros a monitorear:
- Temperatura
- Humedad ambiental
- Presión atmosférica 
- Humedad en la planta

Material:
- Sensor BME280
- Sensor de humedad de tierra SPARKFUN 
- Microcontrolador Argon Particle
